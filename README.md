# helio-landing-page

A simple landing page for a healing app

## Installation steps

1. Clone github repo 
   ### `git clone git@github.com:literallymarvellous/helio-landing-page.git`

2. Change working directory
   ### `cd helio-landing-page`
   
3. Install dependencies
   ### `npm install`
   
4. Run app
   ### `npm run dev`
